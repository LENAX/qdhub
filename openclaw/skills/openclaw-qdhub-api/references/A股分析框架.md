# A 股分析框架（投研结构化参考）

本框架用于规范 A 股投研分析的**宏观 → 中观 → 微观**三层结构，并补充**消息面、基本面预期与情绪分析**。分析时按此框架组织内容，并结合 QDHub 提供的接口获取数据（接口路径见各节说明，未标注的需用 custom-query 或外部数据补充）。

---

## 一、宏观分析

### 1.1 宏观流动性分析

- **市场总体成交量**：全市场或主要指数近期成交额/成交量水平、与前期对比、是否放量/缩量。  
  - 数据来源建议：QDHub 指数 K 线或全市场汇总（如 `/api/v1/analysis/kline` 取指数、或 custom-query 汇总日成交）。

- **净流入流出**：市场整体或主要板块资金净流入/净流出金额及方向变化。  
  - 数据来源建议：`/api/v1/analysis/money-flow`（按 trade_date 或 start_date/end_date、market）。

- **利率**：当前关键利率水平（如政策利率、SHIBOR、国债收益率等）及对估值与流动性的含义。  
  - 数据来源：多需外部数据或 QDHub 中若有利率相关表；无则注明“需另行接入利率数据”。

- **M1-M2 剪刀差**：M1 与 M2 同比增速差，用于观察企业活期与整体流动性变化。  
  - 数据来源：多需宏观数据源；无则注明“需另行接入宏观数据”。

### 1.2 美股指数分析

- **标的**：美股 SPY（标普 500 代表）、QQQ（纳斯达克 100 代表）。
- **分析要点**：近期涨跌方向与幅度；是否形成明确趋势（如均线多头/空头、高低点演变）；对 A 股风险偏好与外资流向的参考意义。
- 数据来源：若 QDHub 有美股或外盘指数数据则用其 K 线；无则注明“美股数据需另行接入”。

### 1.3 A 股主要指数分析

- **标的**：上证指数、深证成指、创业板指、中证 500、中证 1000（对应 ts_code 需在 QDHub 的指数列表中确认）。
- **分析要点**：**涨跌**（近期涨跌幅、相对强弱）；**技术支撑与压力位**（基于 K 线、均线或前高/前低）；**大盘涨跌比**（上涨家数/下跌家数或类似广度指标）；**大盘股与小盘股表现**（上证/沪深300 vs 中证500/1000）。
- 数据来源建议：`/api/v1/analysis/kline`、`/api/v1/analysis/stocks/indicators`、`/api/v1/analysis/limit-stats`、`/api/v1/analysis/limit-up-comparison` 等。

## 二、中观分析

### 2.1 热门题材与板块

- **近期热门题材/板块前十**：按热度或涨幅排序的前十题材/行业或概念。  
  - 数据来源建议：`/api/v1/analysis/concept-heat`、`/api/v1/analysis/sector-limit-up-stats`。

- **资金净流入/流出前十**：按资金净流入或净流出排序的板块/概念前十。  
  - 数据来源建议：`/api/v1/analysis/moneyflow-concept`、`/api/v1/analysis/money-flow`。

- **重点关注**：**持续上榜**（多日出现在热度/资金前十）；**快速上升**（排名或热度、资金流入明显抬升）；**走出技术性趋势的板块指数与 ETF**（均线趋势、突破/跌破）。

### 2.2 量化筛选要点

- **价格与均线**：收盘价持续运行在 5 日线上方且 5 日线持续向上。  
  - 数据来源：`/api/v1/analysis/kline` + `/api/v1/analysis/stocks/indicators`（MA5）或 custom-query。

- **成交量与资金**：成交量持续放大且资金净流入。  
  - 数据来源：`/api/v1/analysis/money-flow`、K 线 volume/amount。

### 2.3 板块内权重与领涨股

- 对上述筛选出的板块，分析其**权重股**与**领涨股**（涨停数、涨幅、成交额排名）。  
- 数据来源建议：`/api/v1/analysis/sector-limit-up-stocks`、`/api/v1/analysis/concept-stocks`、`/api/v1/analysis/limit-up-list`、`/api/v1/analysis/popularity-rank`。

### 2.4 板块与个股选择偏好（高度关注）

在板块和个股的筛选中，对以下特征**高度关注**：

- **价量配合健康**：**温和上涨**（非暴力拉升）、**上涨有量、下跌无量**（涨时放量、跌时缩量），体现筹码稳定、抛压较轻。
- **量能放大与资金方向**：**近期成交量相对前期显著放大**，且**总体资金呈现净流入**；可与前期一段时间的均量或资金流向对比。
- **趋势与均线**：**股价可以持续维持在五日线上且走出上涨趋势**的板块或个股也要重点关注（与 2.2 量化要点一致，此处强调为选股优先方向）。

对符合上述特征的板块，进一步做**消息面核对**：

- **分析该板块是否有新闻利好**：若有明确政策、行业或公司利好，可纳入重点关注并与其他维度交叉验证。
- **即使当前没有明显新闻利好，也要高度关注**：此类“温和上涨 + 放量 + 资金净流入 + 站稳五日线”的组合，常是**资金秘密买入、建仓**的信号，**隐含市场尚未充分反映或尚未公开的利好**；可**深入挖掘公开信息**（公告、研报、行业动态、产业链变化等）**寻找逻辑和证据**，提前跟踪潜在催化剂。

数据来源：K 线（量价）、`/api/v1/analysis/money-flow`、`/api/v1/analysis/moneyflow-concept`、`/api/v1/analysis/stocks/indicators`（MA5）、`/api/v1/analysis/news` 等。

## 三、微观分析

在宏观与中观结论基础上，对**板块龙头与领涨股**做个股级分析。

### 3.1 技术面

- **技术支撑与压力位**：基于 K 线、均线、前高/前低给出关键支撑与压力位。
- **技术形态**：如突破、回踩、整理、头肩等形态描述（若有）。
- **价格趋势**：短期与中期趋势方向（均线多头/空头、高低点演变）。
- 数据来源：`/api/v1/analysis/kline`、`/api/v1/analysis/stocks/indicators`。

### 3.2 资金与量能

- **资金流入/流出**：个股或同板块口径下的净流入流出及变化。  
  - 数据来源：`/api/v1/analysis/money-flow`（ts_code）。

- **成交量**：是否出现**密集放量**或持续放量，与价格走势的配合情况。  
  - 数据来源：K 线 volume/amount。

### 3.3 技术分析要点（通用）

技术分析**八字诀**：**支撑压力、斐波那契**。资金的买入卖出行为会反映到股价上，股价在**关键支撑位和压力位**的行为具有重要的技术分析意义，**成交量**可为分析提供关键依据；技术分析须**结合量价行为综合判断**。以下适用于板块指数与个股，中观、微观分析均可参照。

- **周期**：**长周期（约一个月）的 K 线形态**有助于综合判断多空力量；短周期形态参考价值有限。

- **关键形态**：
  - **双突破**（同时突破支撑、压力与趋势线）是重要的技术形态，需重点识别。
  - **大波动率价格区间向小波动率区间运动**，且价格逼近关键支撑位或压力位时，也值得高度关注。
  - **斐波那契与黄金比例**：斐波那契比例及**黄金 0.618** 常可用来计算精确的回调价格。将区间内**最高价与最低价**用斐波那契工具划分，可得到回调位，价格往往落在计算位附近，可据此预判支撑/压力与加减仓区域。

- **技术指标**（仅以下有用，其余可不看）：
  - **均线**：**5 日、10 日均线**辅助短期趋势；**20 日、60 日均线**辅助中长期趋势。
  - **成交量**：与价格配合判断放量/缩量、量价背离等。
  - **ATR**：用于衡量**波动与风险**，也可辅助**仓位管理**。  
  数据来源：`/api/v1/analysis/stocks/indicators`（MA、可扩展 ATR）、K 线 volume；ATR 若接口未提供可用 custom-query 或本地计算。

- **形态分析**：
  - **长周期区间形态**（有用）：**双突破**、**楔形**、**茶杯带柄**、**W 底**、**M 头**、**岛形反转**、**N 字反转**、**弧形底**等。
  - **短周期形态**（基本没用）：如红三兵、三只乌鸦、射击之星等，可不依赖。
  - **单根 K 线**：仅在**关键支撑/压力位**时，可适当参考**仙人指路、pinbar** 等单根 K 线形态，其余意义不大。

### 3.4 输出要求

- 微观结论须与中观、宏观相呼应；若给出标的或建议，须遵守技能中的**风险提示**与**止损点**要求。

## 四、消息面分析

- **从新闻入手**，识别具有**大级别影响力**的事件与新闻：至少能显著影响一家公司、一个行业、一国或全球，或在社交媒体引发大量关注的话题。
- **事件分类**：
  - **突发性事件**：当日或短期爆发的利空/利好；
  - **未来利好/利空事件**：有明确时间或节奏的预期类事件。
- **分析要点**：
  - 对**上市公司个股 / 行业 / 大盘**的影响方向与幅度；
  - 事件的**持续性**（一次性冲击 vs 持续发酵）；
  - 若对个股/行业/大盘可能造成**显著基本面预期改变或冲击**，列入**重点关注事件列表**，并在此基础上制定交易计划。
- **重点关注**：**未来利好 + 持续发酵**的事件优先关注。
- **交易节奏参考**（须结合风险提示与止损，不构成投资建议）：
  - **股价尚未反映预期**：可考虑在股价**关键支撑位**附近买入潜伏；
  - **股价开始反映利好、但未出现大幅上涨**（如未出现单波 20%–30% 级别拉升）：可结合支撑压力与股价位置、根据风险承受能力考虑追击；若出现涨停，可允许打板参与；
  - **股价已处于高位**（如连续涨停）：优先在**断板、爆量、分歧**之后，若股价仍能稳定、不跌破趋势，再考虑低吸介入。

## 五、基本面预期分析

- 在消息面与宏观/中观数据基础上，判断事件与数据对**盈利、估值、景气度**的预期影响。
- 将**能显著改变基本面预期或造成冲击**的事件/数据纳入重点关注，并与消息面中的“重点关注事件列表”和交易计划衔接。
- 若 QDHub 有财务、业绩相关接口，可结合 `/api/v1/analysis/financial/indicators`、财报表等做预期修正；无则注明数据来源或局限。

## 六、情绪分析（A 股短线与风控参考）

A 股情绪分析是短线交易和风险控制的重要一环。分析时需结合涨跌停统计、人气榜、连板高度、龙头表现等数据（如 `/api/v1/analysis/limit-up-ladder`、`limit-up-list`、`popularity-rank`、`limit-stats`）。

### 6.1 大周期（经验上约 1–1.5 个月一轮）

| 阶段 | 特征 |
|------|------|
| **无序震荡期** | 题材高速轮动，无持续性，属于“垃圾时间”，最好不交易或仅用很小仓位试水。 |
| **题材发酵期** | 资金开始主动进攻少数有显著利好催化的板块。 |
| **题材主线主升期** | 出现龙头，连板高度突破 7 板；可选有换手量的标的或市值较大的容量票；买点可为突破前高后追入，或爆量分歧且次日不破位时低吸。 |
| **高位震荡期** | 龙头高度维持在 10 板以上但不再连板，进入高位震荡；**只能弱分歧时低吸、转一致时高抛，仓位不可过重**。 |
| **题材退潮期** | 龙头跌停，大盘与绝大部分个股跟跌，赚钱效应极差；**龙头跌停即视为行情结束，禁止开仓，空仓休息**。退潮后往往有数月的无序震荡期。 |

### 6.2 情绪节点（题材内部）

- **发酵**：题材出现首日出现多个涨停，板块部分个股跟涨。
- **分化**：领涨股继续涨停，跟风股高开低走。
- **分歧**：领涨股高开低走、断板套人。
- **退潮**：无资金继续接力，题材行情结束，资金转战其它题材。

当**龙头打出辨识度**、出现**高人气和持续讨论**时，可能进入**主升期**，龙头连续涨停，此时节点为**主升**。主升后仍可能转入**分歧**：若是**弱分歧**，可于**当日低吸博弈**；若是**强分歧**（如大跌 6% 甚至更多，且没有强烈反弹），则要**等到第二天**再观察或考虑介入。

**买点规则（极其重要）**：

- **打板**：**只在题材强度确认的初期**打板，不在分化或高位盲目打板。
- **龙头分歧低吸买点**：**只在龙头的弱分歧买入**；
- 分化日**没有买点**；
- **弱分歧当天买**，**强分歧等第二天**再定。

### 6.2.1 情绪判断与题材内选股法则

**情绪如何判断**：**情绪标和龙头的走向**对板块、甚至市场整体都有重大影响。**情绪标和龙头的竞价涨停资金单量**（隔夜封单、集合竞价封单等）可以帮助判断板块和龙头的情绪强弱。

- **主升期龙头的隔日竞价预期**：正常预期是竞价涨停金额达到前日的**1.5 倍–2 倍**。**低于此即为不及预期**，典型表现是**一字开板但会炸板、涨停无法封死**。此时必须**关注资金承接力度**：承接力度够，可能是分歧买点；承接不够则容易吃大面，**不可买**。**若竞价涨停封单超预期**（明显高于 1.5–2 倍或显著强于前日），则要**高度关注该龙头和题材板块**，可按买入规则去**抢最强或次强的标的**；实在买不到就**放弃**，不追跟风。

- **买题材内个股的法则**：**能买龙头就买龙头**；买不到龙头则买**次强的换手涨停票**，或**容量中军**（走强趋势且流通市值较大的标的），或**创业板套利票**；若龙头、次强、中军、创业板套利都买不到，则**放弃**，不勉强追跟风。

- **情绪极好且流动性充足的特征**：板块出现**多个创业板 20% 涨停**，甚至出现**几个北交所 30% 涨停**的弹性票，说明资金进攻意愿强烈，当天追**涨幅前排**和**高辨识度票**风险相对较小、收益弹性较大。

- **实时盘口与承接判断**：可关注**实时盘口的成交量、资金流入和 VWAP 线**。若价格**跌破 VWAP（或关键均线/分时支撑）且不拉回**，则判断**承接不足**，**不能参与**。

- **交易与开盘价的黄金法则**：**开盘价是多空的交界线**。价格**高于开盘价**尽量做多；价格**低于开盘价**尽量做空或**不做**（不逆势抄底、不勉强做多）。

### 6.3 主升与退潮判断

- **主升期**：若出现影响力持续发酵的题材，板块维度上可见新闻利好持续放出，题材在人气榜/涨幅榜维持高排名；板块领涨股人气与资金买入持续加强，走出龙头地位；出现多个情绪标、连续一字板行情。此时可考虑有换手量的标的或市值较大的容量票；买点可为突破前高后追入，或爆量分歧且次日不破位时低吸。
- **退潮判断**：**龙头断板 + 次日跌停** → 板块情绪进入**退潮期**；**龙头断板但次日不跌停** → 板块情绪进入**高位震荡期**。

### 6.4 龙头股持仓与风控原则

- **有利润时**：可持有至出现**跌停**或股价跌至**初始止损价**附近再止损。
- **高位震荡期**：仅可在**弱分歧时低吸、转一致时高抛**，仓位不可过重。
- **龙头跌停**：视为行情结束，**禁止开仓，空仓休息**。
- **退潮后**：往往有数月的无序震荡期，属交易垃圾时间，**最好不交易或仅极小仓位练习**。

（以上情绪与交易节奏仅供分析框架参考，不构成投资建议；输出时须遵守技能中的风险提示与止损点要求。）
