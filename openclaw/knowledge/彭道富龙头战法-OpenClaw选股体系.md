# 彭道富龙头战法精华 — OpenClaw 选股体系文档

**用途**：供 OpenClaw 在 A 股选股/复盘时作为**方法论约束**与**检查清单**；须与 **QDHub 实时数据**交叉验证，不得替代独立判断。  
**来源摘要**：彭道富《股市极客思考录》《龙头信仰》及公开课程/访谈中反复阐述的框架（「龙头四维」「一花三叶」「破执」等）。  
**合规声明**：本文仅为知识整理，**不构成投资建议**；市场有风险，过往框架不保证未来有效。

---

## 一、先立三观：龙头战法不是什么

在执行任何选股规则前，Agent 与用户应统一认知（避免把「龙头」口号化）：

| 常见误解 | 彭氏强调的正解 |
|----------|----------------|
| 谁涨得高谁就是龙头 | **用过程与逻辑定义龙头**，而非用结果定义；「高」可能是虚高 |
| 龙头战法 = 打板 / 追涨停 | **领涨与是否涨停无必然关系**；打板只是部分场景下的手段 |
| 龙头战法 = 纯技术分析 | 本质是**博弈分析**：逻辑、基本面、题材、定性、强弱等多维共振 |
| 见暴涨就冲 | 需 **价值加持 + 取舍**；不符合模式的机会应主动放弃 |
| 信仰 = 口号 | **正信**：取舍、原则、抵御诱惑、过程正确；不是盲信 |

**一句话**：龙头战法是「**如何选择行情、价值、级别与取舍**」的选股哲学，不是单一买点技巧。

---

## 二、「一花三叶」：三条路径（选股前先定赛道）

同一套龙头哲学，在 A 股常见三种落地方式；OpenClaw 应先判断用户/任务属于哪一脉，再选用不同数据权重：

| 路径 | 别称 | 核心特征 | 典型持有周期 | QDHub 侧重数据（示例） |
|------|------|----------|--------------|------------------------|
| **股权龙头** | 价值龙头 | 基本面「一览众山小」，宜当**股权**看待 | 长 | 财务、行业地位、估值与业绩趋势 |
| **白马龙头** | 价值投机 / 中间路线 | **有业绩/有故事/有事件**，盘面与价值共振，吃主升浪波段 | 中 | 业绩预期 + 题材 + 量价与资金 |
| **黑马龙头** | 盘面龙头 / 人气龙头 | 题材、情绪、板块效应、稀缺与逼空 | 短 | 涨跌停结构、概念梯队、换手与情绪 |

**融合原则**：三脉**同气同宗**——都是找「同类中的王者」，区别在时间与价值锚定不同；同一标的在不同阶段可能在三脉间迁移。

## 三、核心定义：龙头四维（选股检查的基石）

彭道富将「真龙头」收敛为四个维度；**四维不张，龙头不降**——OpenClaw 可将每一维转为可问答的布尔/打分项。

### 3.1 价值加持（定海神针）

- **业绩是价值之母**（不等于照搬价值投资）：龙头战法同样强调最终要**能往业绩上靠**——涨价、放量、订单、行业复苏等带来的**盈利弹性**。
- 区分 **已兑现** 与 **预期**：题材可以炒预期，但真龙头往往存在「能把故事讲回业绩」的路径。
- **反面**：长期无法落脚到任何价值与逻辑链条的纯情绪博弈，偏离「正朔龙头」。

**OpenClaw 检查句**：

- 该标的上涨理由，能否分解为「价」或「量」或对业绩的**可验证路径**？
- 若仅概念无业绩锚，是否应降级为高风险博弈并缩小仓位或放弃？

### 3.2 对外影响力（级别）

龙头应对外「带节奏」，而非孤立独舞：

1. **对大盘**：强龙头常能**牵引指数或风险偏好**（级别高）；弱龙头与大盘脱钩则级别有限。  
2. **对板块**：上涨能**传染**同链/同题材标的，形成**板块效应**。  
3. **对逻辑与路径**：定义行情演绎路径（如行情沿某叙事扩散），后来者模仿其结构。

**OpenClaw 检查句**：

- 该股走强时，**板块内跟风**是否同步？若无板块效应，是否更像「独狼」而非龙头？
- 该叙事是否是**当前市场主线级别**？（需结合指数环境、涨停分布、概念持续性）

### 3.3 市场性（裁判权在市场）

- 你认为的价值与市场给出的**定价与资金选择**是否一致？  
- **不预测未来**：龙头战法强调处理当下——在**已走强的结构**中识别领涨，而非纯潜伏臆测。

**OpenClaw 检查句**：

- QDHub 是否显示**资金、涨幅、封单/换手**等对叙事有「投票」行为？
- 若逻辑自洽但市场不认，是否应**等待市场确认**而非左侧硬扛？

### 3.4 第一性（时间与空间上的「第一」）

- **时间第一性**：率先启动、率先表态（黑马语境下常体现为率先涨停/引领节奏）。  
- **空间第一性**：同梯队中**涨幅/空间预期**领先。  
- **地位第一性**：叙事中的「老大」——最硬逻辑、最核心受益、最高辨识度。

**OpenClaw 检查句**：

- 在同题材中，它是**辨识度最高**的那一个吗？还是后排跟风？
- 「第一」是否建立在**价值与影响力**之上，而非单纯先涨停？

## 四、破执清单（过滤伪龙头与错误手法）

在输出候选池前，应显式排除以下**认知陷阱**：

1. **破「结果论」**：不能只因涨得多就封为龙头。  
2. **破「唯涨停论」**：涨停是表象，**领涨与带动**才是核心。  
3. **破「形态/K 线至上」**：技术工具要用，但不能被形态绑架；压力位对真龙头常「钝化」，但仍需结合位置与透支度。  
4. **破「必须打板」**：买点是次要矛盾，**选对龙、选对级别**优先。  
5. **破「全天候作战」**：周期不匹配时应**空仓/降频**（与「取舍」一致）。

---

## 五、节奏与风控（Agent 输出结论时的必备语境）

### 5.1 周期与位置

- **性质决定高低**，而非涨幅数字本身；警惕**情绪末端**与**逻辑透支**。  
- 龙头在退潮期可能**领涨变领跌**；需区分「龙头见顶」与「市场系统性退潮」。  
- **少出手**：模式内机会稀缺；多数时间等待优于强行交易（与资金博弈现实一致，非彭氏独论，但与「取舍」相合）。

### 5.2 真假龙头

- **一佛生，百魔出**：龙头越热，假龙头越多。  
- 求真路径：**从价值与领涨属性验证**，而非仅图形；需大级别事件/硬逻辑背书（程度因三脉而异）。

### 5.3 输出要求（对齐 OpenClaw 技能）

若向用户给出标的，须保留：**推理链（数据→推断）**、**风险提示**、**止损思路（仅供参考）**，并声明不构成投资建议。

## 六、OpenClaw 可执行流程（建议）

以下为 **KISS** 工作流，便于 Agent 复用；可与 `如何把握节点型机会-结构化笔记.md` 等情绪节点文献对照（注意不同老师对「分歧/分化」买点表述可能不同，勿混用）。

```
1) 定脉：股权 / 白马 / 黑马 —— 决定持有周期与数据权重
2) 定级：是否主线、指数环境是否配合 —— 决定仓位与预期
3) 四维打分：价值加持、影响力、市场性、第一性 —— 每项通过/待验证/不通过
4) 破执过滤：结果论、独狼无板块、纯情绪无价值锚 等 —— 剔除伪龙头
5) QDHub 验证：涨停梯队、概念成分、资金、基本面钩子 —— 证据化
6) 输出：候选 + 反证条件（何种数据出现即推翻龙头假设）
```
