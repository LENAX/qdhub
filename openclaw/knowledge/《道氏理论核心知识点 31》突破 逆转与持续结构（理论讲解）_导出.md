## 二、下跌过程中的形态演变

- 一般先从下跌开始：会做大的**顶部结构**（如头肩结构、顶部结构）。
- **顶部形成的标志**：头肩结构**跌破颈线**（脖子），跌破颈线后**回抽无力**，再次下跌，就标志着顶部形成。
- 顶部形成之后：
  - **位置①**：盘整剧烈（刚跌，抵抗剧烈），做大的**中继形态**（如箱体）；发现涨不上去，箱体处波动率变小；再次跌破。
  - **位置②**：做**三角形**；此处抵抗非常剧烈，是波动率大的中继形态。
  - **位置③**：又来中继形态，抵抗稍弱的中继形态；再次跌破；有时走小通道；最后暴跌，**行情彻底跌完**——即下跌通道，最后加速。
- 形态学的运用：顶部结构（可以是头肩，也可以是倒三角形、菱形）→ 跌破颈线（**颈线概念非常重要**）→ 回抽 → 做大形态/中继形态（所学的中继形态）→ 再中继形态 → 下跌通道加速。
- 概括：先**突破**（跌破颈线等），行情发生**逆转**，逆转完后**持续**（中继形态即持续结构）；这是一波下跌行情，到底之后进入下一阶段。

## 三、趋势反转与持续的关键信号（底部之后）

- 到底之后：会在底部做大的盘整（如箱体，或例如菱形）。
- 以菱形为例：做菱形，**菱形上下边缘、中轴突破** → 突破后回抽得到确认 → 确认后继续。
- **突破** = 行情的逆转的突破；突破后若又回来，则尚未逆转。**回抽得到确认的位置 = 发生逆转**。此处突破、回抽得到确认，形成 N 字形，就得到**持续**；行情继续走。
- 随后：可走箱体，再往上走三角形，再往上行情一波一波上去。
- 行情的结构永远如此：
  - 先顶部结构 → 跌破颈线、回抽无力、下跌 → 走持续结构（箱体、大/小箱体、通道）→ 再跌 → 走三角形 → 三角形跌破 → 下跌通道（有时没有也无所谓，有时跌破又拉起来）→ 跌破后继续。
  - 到底部再做盘整 → 关键位置**突破** → **突破回抽得到确认时，就是逆转** → 逆转后再往上突破 N 次、再做形态，就是**持续**。
- 几乎所有行情都是这样：一轮一轮「顶部结构 → 突破、逆转、持续」到下面，「底部结构 → 突破、逆转、持续」再往上面。不管一分钟、五分钟、一小时 K 线还是日线，都可以看到大量这种结构，几乎到处都是。
- 因此可把它叫做金融交易技术分析的 **DNA**，即最小的分子结构。**只要掌握好这个结构，对整个市场的结构就会非常清晰。**

## 四、构建交易系统的实战应用（买点）

- 掌握这些知识之后，什么时候买入（股票或品种）？在**三个地方买**：**买突破、买逆转、买持续**。
  - 突破的时候，看情况可以买；
  - 突破回抽得到确认的时候，也可以买；
  - 持续的时候（即中继盘整的时候），也可以买。
- **交易的秘密就是在这几个点做文章：突破、回抽之后、持续的时候。**
- 细节上大家会不一样；威科夫理论、缠论里的二买三买，全部是在这些地方做文章。学到最后殊途同归，**把这几核心点掌握好，就能构建一套好的交易系统，找到好的方法。**
- 以上就是突破、逆转与持续结构；在 K 线里会再展示。
