# 如何把握节点型机会 — 结构化笔记

**来源**: 江Neo · B站视频《三小时制作！教你如何把握节点型机会》

> **个人标注**：本笔记中关于「分化日/分化次日买入」为**原作者观点，仅供参考**。本人体系：**买在弱分歧节点，不买在分化节点**。此处仅作对照与辨识用。

---

## 一、情绪节点的判断

### 1.1 情绪启动与转强的预判

| 维度 | 要点 |
|------|------|
| **时间锚点** | 如：假期结束次日（例：五月六号）常是情绪转折点；周末有题材发酵时，周一情绪易转强。 |
| **先行信号** | 人气榜标的飙到第一、题材明牌 → 情绪即将转强。 |
| **杀跌背景** | 若节前/前几日杀跌时间很长，节点起的新题材往往让情绪延续时间更长。 |

### 1.2 分化节点 vs 杀跌日（小周期结束）

- **分化节点**：高位连板变少、首板很多 → **情绪并未结束**，只是小分化。
- **判断依据**：获利盘要兑现；情绪火过一天后，次日容易分化。
- **前瞻判断**：**前一天即可判断**“明天会分化”，不必等开盘。例如：
  - 核心标的一字/缩量顶到高位 → 次日大概率分化；
  - 四板、五板后获利盘集中 → 分化日。

### 1.3 竞价与盘口对节点的验证

- **竞价开得太低**（如成飞、立军三板/四板时竞价直接开低）→ 当天**一定是分化日**。
- **核心标的竞价强度不够**（封单/分单量明显掉）→ 分化日确认。
- **锚定方式**：用**核心个股的放量节奏**锚定整体情绪节奏（如成飞集成作为市场核心时，直接锚定成飞即可）。

### 1.4 情绪周期尾段（穿越期）

- 情绪火热约 1～2 周后**不会立刻结束**，会有“穿越”效应。
- 亏钱效应是**慢慢积累**的，赚钱效应的消退也是渐进的，通常还会走 3～4 天（行情特别大时可能穿越再穿越）。
- **有主流题材 + 情绪好**时：分化后常能接着往上，核心不会轻易 A 杀；**除非涨到没有大资金敢买**。

## 二、交易节奏的把握

### 2.1 买在“分化”，卖在“一致”（原作者观点）

- **分歧买入、一致卖出**的量化含义（**仅供参考**）：
  - **买**：原作者在情绪**主升期的小分化点**买（分化当天或分化次日）。**本人体系：买在弱分歧节点，不买在分化节点。**
  - **卖**：在情绪**高潮、获利盘很多**时卖。

### 2.2 分化日 = 上车日（原作者观点，仅供参考）

- 原作者认为：**分化当天或分化次日** = 买入有预期标的的**最佳时机**。**本人体系：买在弱分歧节点，不买在分化节点。**
- 原作者所述分化日参与方式（仅作对照）：
  - **直接参与核心**（头铁顶核心），或  
  - **等核心确认后再买**（例如做换手龙、一进二）。

### 2.3 核心标的的卖出节奏（重要）

| 规则 | 说明 |
|------|------|
| **核心放量的后一天卖** | 不要在核心**放量当天**卖，而是**放量次日冲高卖** → 容易卖在最高点。 |
| **核心多活一天** | 核心在放量之后**通常还能多活一天**；若不确定是否核心，可参考“放量后一天再卖”。 |

### 2.4 按“小周期”选股与持仓

- 若按**“从小周期分化到下一次分化”**的节奏做：
  - **底部换手少**的标的 → 顶部换手会多、上涨压力更大，更适合做**短一段**（如到下一次分化）。
  - **底部换手多**的标的 → 可能走得更远，但持股节奏不同（更多换手、不一定拿得住）。
- **个股的放量节奏与情绪的节奏是对应起来的**（连板如此；抱团则看个股小周期）。

### 2.5 不同阶段做不同事

- **适合走连板时** → 做连板（拐点参与连板性价比最高）。
- **情绪余温、连板无新题材但活跃度还在** → 做**抱团**（换手高、底子好、有人气的核心）。
- **分化节点上出现新题材**且**尚无亏钱效应** → 原作者认为**一定要参与**（**仅供参考；本人体系：买在弱分歧节点，不买在分化节点**）；若有明显亏钱效应，最好等**亏钱效应次日**再考虑。

### 2.6 全放量日的节奏

- **绝大多数情况**：标的**全放量日应考虑卖出**。
- **例外**：若判断**未来情绪还会继续强**，可保留或在该日参与；此时能否封住/回封，很大程度取决于**资金对后续情绪的预期**。

## 三、方法（复盘 → 临盘 → 选股 → 参与方式）

### 3.1 复盘到临盘的整体流程

1. **复盘**（前一晚）  
   - 分析：第二天有没有预期、未来还能怎样、处于哪个赚钱阶段、明牌还是不明牌、题材会不会发酵、**当前情绪节点是否合适**。  
   - 输出：**把合适高度的标的筛出来**（绝大多数标的可排除，真正能买的约 5%～10%）。

2. **临盘**（当日）  
   - 看筛出的标的**有没有资金真的愿意进攻**。  
   - 若上板：判断是**一次性封板**还是**会炸板**。  
     - 一次性封板 → 直接扫。  
     - 会炸板 → 等**快要最后一封时**再进。

### 3.2 炸板是不是“机会”的判断

| 观察项 | 含义 |
|--------|------|
| 竞价分单量 | 足够大（如 60 万手级别）→ 资金承接意愿强。 |
| 回封次数与力度 | **多次回封** = 有资金愿意回封 = 认为还能涨。 |
| 情绪与高度 | 情绪在向好 + 个股只有一板（首板）→ 炸板后回封可视为机会。 |
| 人气与强度 | 人气榜前排、当日资金流入强度够 → 加分。 |

结论：**炸板后回封 = 机会**，前提是情绪向好、个股高度不高、盘口有承接。

### 3.3 板块与题材的筛选

- **板块**：**新板块 + 有事件 + 情绪节点好** → 板块内才有合适标的。  
- **排除**：大成交额（如 17 亿）酌情排除、无权限的 300、**炒过的老龙反抽**（预期不多）。  
- **新题材**：若不知道未来预期，看**次日板块核心标的强度**、是否有**容量标**；强度不够则板块整体一般。

### 3.4 一进二：买谁、何时买

- **原则**：能买到哪个买哪个，但**必须能封得住**。  
- **盘口要点**：  
  - **前 5 分钟换手速度**要够。  
  - 成交量收量后再次上板（**左侧盘中拉升**）。  
  - 当日**资金流入**更佳。  
- **回封可打**：左侧板上换手（约 20%）、**大单出来时**往往是唯一再进场机会。

### 3.5 用“预期”反推（盘口验证）

- **没人愿意打回封** → 预期不足（要么个股边际不够，要么对情绪预期不好）。  
- **有人愿意一直在板上买** → 有预期；可从**盘口反推**它有没有预期。  
- 若学过“预期内因”：**预期 + 盘口结合**更好——既知道有预期，盘口又验证预期。

### 3.6 分化日的三种参与方式（原作者观点，仅供参考）

> **本人标注**：买在弱分歧节点，不买在分化节点。以下为原作者所述分化日参与方式，仅作对照。

1. **直接顶核心**：风险大，适合已有仓位或特别确定时。  
2. **等核心分稳后做换手龙**：如天箭、力源——**没有直接顶一字**很重要；一字能分稳，换手就还有预期。  
3. **同板块 20cm/30cm 套利**：**10cm 强度足够**时，可到 20cm、30cm 套利。

### 3.7 全放量板与抱团

- **全放量板**：  
  - 多数情况慎买；若买，需能判断**未来情绪还会强**。  
  - 没人封板常见原因：个股边际不够，或**对未来情绪预期不好**；有预期、有利差后资金会回流。  
- **底子好的标的**：更容易抱团、也更容易有预期（如封力比、流通盘更利于锁单）。  
- **抱团**：可视为一种“题材”——**只要有资金买即可**；做**换手高、底子好、有人气**的核心。

### 3.8 卖点与细节

- **9:35～9:40**：若竞价已决定要卖，很多标的会有拉升，**此时卖往往较好**。  
- **防买在高点**：要考虑**拉不上去**的情况，否则容易买在当天高点。  
- **模式随情绪变**：先有思路 → 把个股分析好 → 第二天临盘看盘口，**这一套**贯穿始终。

## 四、小结：节点型机会三句话

1. **情绪节点**：用“启动/分化/穿越”划分小周期，用核心标的放量节奏锚定；分化日可前一日预判，竞价与盘口验证。  
2. **交易节奏**：原作者为分化日（或次日）买、一致/高潮卖（**仅供参考；本人：买在弱分歧节点，不买在分化节点**）；核心放量**后一天**卖；不同阶段做连板或抱团，全放量多数以卖为主。  
3. **方法**：复盘筛出少数标的 → 临盘看资金是否进攻、封板还炸板 → 炸板看回封与预期，一进二看换手与回封；原作者在分化日参与核心或换手龙或 20/30cm 套利（**本人体系：买在弱分歧节点，不买在分化节点**）；用盘口反推预期，底子好、有人气的更适合抱团。
