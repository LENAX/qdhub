## Phase 3：复盘报告系统（当前主线）

> 优先级：S（必须先做）/ A（核心交付）/ B（增强）

### 3.1 数据与口径底座（S）

- [ ]**统一复盘口径**：指数、市场情绪、消息面、资金面、赚钱效应、龙头跟踪的字段字典与计算口径
- [ ]**交易日快照层**：按交易日固化 EOD 快照，支持近 3/5/10/20 日综合分析
- [ ]**维度评分规则**：有无、强弱、持续性、分布、风格的规则化量化

### 3.2 复盘报告 MVP（A）

- [ ]**日报模板**：指数 + 情绪 + 消息 + 资金 + 赚钱效应 + 龙头跟踪六大模块
- [ ]**多日综合分析**：区间趋势、拐点、持续性判断
- [ ]**龙头跟踪看板**：候选池、晋级/掉队、连板/换手/资金背离
- [ ]**报告产出能力**：页面查看 + Markdown 导出

### 3.3 复盘训练扩展（B）

- [ ]**题材周期复盘和模拟训练**：回到历史任意一天，提供当日指数、板块、消息、资金、个股、涨停数据，支持训练选股与交易并复盘龙头

## Phase 4：因子分析研究（Jupyter -> QDHub）

> 优先级：S（必须先做）/ A（核心交付）/ B（增强）

### 4.1 Jupyter 因子研究标准化（S）

- [ ]**研究模板标准化**：因子定义、去极值、标准化、中性化、分层回测、IC/IR、换手与交易成本
- [ ]**可复现研究流程**：统一输入输出与实验记录格式

### 4.2 因子迁移到 QDHub（A）

- [ ]**因子元数据管理**：名称、表达式、版本、依赖字段、适用频率
- [ ]**离线批计算与查询**：先日频、可回溯，不先做全实时
- [ ]**因子评估看板**：IC 曲线、分组收益、稳定性统计

### 4.3 后续增强（B）

- [ ]**跨市场与跨周期因子稳定性分析**
- [ ]**因子组合自动化实验框架**

## Phase 5：选股系统（参考同花顺 / 果仁）

> 优先级：S（必须先做）/ A（核心交付）/ B（增强）

### 5.1 选股基础能力（S）

- [ ]**统一股票池构建器**：市场/行业/题材/指数成分 + 基础约束过滤
- [ ]**多维指标与因子排序器**：支持权重配置与模板保存

### 5.2 选股 MVP（A）

- [ ]**多条件筛选系统**：阈值、区间、排除规则（ST/停牌/流动性不足等）
- [ ]**候选清单管理**：入选原因、因子暴露、与复盘结论联动
- [ ]**策略模板化**：首板接力 / 双突破 / 龙头首阴等 playbook 的规则化配置

### 5.3 后续增强（B）

- [ ]**选股结果归因分析**：命中率、超额收益、回撤归因
- [ ]**盘中增量更新与预警**

## Phase 6：交易记录与复盘系统（新增重点）

> 优先级：S（必须先做）/ A（核心交付）/ B（增强）

### 6.1 交易记录底座（S）

- [ ]**账户体系**：支持多账户（模拟/实盘映射）、账户资产快照、账户维度统计
- [ ]**交易标的管理**：股票/ETF/指数衍生标的统一主数据
- [ ]**交易记录管理**：买卖/撤单/成交记录，含时间、价格、数量、费用、来源策略
- [ ]**盈亏核算引擎**：已实现盈亏、浮动盈亏、手续费、滑点、净值口径统一

### 6.2 交易分析与管理（A）

- [ ]**盈亏曲线分析**：账户净值曲线、回撤、胜率、盈亏比、持仓周期分析
- [ ]**交易计划模块**：开盘前计划、入场条件、风控条件、退出条件
- [ ]**交易复盘模块**：计划 vs 执行偏差分析，单笔与组合复盘
- [ ]**交易日志系统**：盘前/盘中/盘后日志，支持结构化标签与检索

### 6.3 后续增强（B）

- [ ]**AI 交易教练**：基于交易计划、执行与结果给出一致性评估与改进建议
- [ ]**交易行为画像**：识别冲动交易、过度交易、风控违规模式

## Phase 7：策略与交易系统（因子到交易闭环）

> 优先级：S（必须先做）/ A（核心交付）/ B（增强）

### 7.1 策略系统 MVP（S/A）

- [ ]**策略配置框架**：信号、仓位、风控、执行规则可配置
- [ ]**回测引擎**：日频先行（含手续费、滑点、涨跌停与停牌约束）
- [ ]**模拟自动交易**：信号 -> 订单 -> 成交 -> 持仓 -> 绩效闭环

### 7.2 重点体系实现（A）

- [ ]**彭道富体系**：四维/三脉框架映射、策略模板化与验证

### 7.3 后续增强（B）

- [ ]**类似果仁的策略原型开发平台**
- [ ]**策略集成评估与组合层风控**

## 下一阶段：模型研究（准备进入）

研究框架与文献索引见 **[《交易模型研究框架指南》](doc/spec/trading-model-research-framework.md)**（含宏观—中观—微观 × 择时—选股—交易、管线约定、参考文献与 `research/` 基线代码）。

在数据与基建相对稳定后，进入系统化模型研究，方向包括但不限于：

- [ ]**宏观择时模型**
- [ ]**题材 / 板块选择与择时模型**
- [ ]**情绪周期与情绪节点分析、预判模型**
- [ ]**龙头选择模型**（规格草案：[doc/spec/leader-stock-four-dimensions-peng-daofu.md](doc/spec/leader-stock-four-dimensions-peng-daofu.md) — 彭道富「四维×三脉」与 QDHub 特征映射）
- [ ]**近 5 年龙头周期分析与复盘课题**：对比大盘指数周期，梳理历史大题材与关键事件，定位各阶段龙头，复盘完整周期，并生成标准化复盘报告与结构化数据集。
- [ ]**ETF 择时与轮动模型**
- [ ]**总仓风控与择时模型**
- [ ]**ETF / 龙头高波动再平衡模型**
